# Amy Jaffe Nutrition

The independent website for Amy Jaffe Nutrition, rebuilt from the former Wix site.

## Development

```bash
npm install
npm run dev
```

Run `npm test` to create a production build and verify the rendered homepage.

## Asset sources

- The hero uses Mixkit’s [Purple flowers moving with the wind](https://mixkit.co/free-stock-video/purple-flowers-moving-with-the-wind-1878/) under the Mixkit Stock Video Free License. The local copy is retimed to 1.6× its original duration for gentler movement.
